# 🍔 Burger Sales Forecasting using Deep Learning (LSTM)

## Project Overview

This project develops a deep learning model to forecast burger sales using historical sales and weather data. The workflow includes data exploration, preprocessing, feature engineering, baseline machine learning, LSTM model development, evaluation, and deployment with AWS SageMaker.

## Objectives

- Explore the relationship between weather and burger sales.
- Perform exploratory data analysis (EDA).
- Build a LightGBM baseline model.
- Develop an LSTM forecasting model.
- Forecast next-day burger sales.
- Deploy the trained model using AWS SageMaker.

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras
- LightGBM
- AWS SageMaker
- Amazon S3
